## 介绍

`sort()`，按给定方法排序。此方法会改变原数组

[MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/sort)

## 语法

```js
arr.sort(compareFn?)
```

### 参数

<dl>
  <dt><code>compareFn</code></dt>
  <dd>排序函数。参数有二：</dd>
  <dl>
    <dt><code>a</code></dt>
    <dd>第一个元素</dd>
    <dt><code>b</code></dt>
    <dd>第二个元素</dd>
  </dl>
</dl>

### 返回值

排序后的数组
