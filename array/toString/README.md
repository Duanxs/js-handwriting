## 介绍

`toString()`， 数组转字符串。数组中的元素将使用各自的 `toString` 方法转成字符串，随后将所有元素用连接符链接并返回。

[MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/toString)

## 语法

```js
arr.toString()
```

### 参数

无

### 返回值

拼接后的字符串
