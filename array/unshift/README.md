## 介绍

`unshift()`，用以将元素追加到开头，并返回新长度。此方法会修改原数组

[MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/unshift)

## 语法

```js
arr.unshift(item1?, item2?, ..., itemN?)
```

### 参数

<dl>
  <dt><code>itemN</code>(可选)</dt>
  <dd>追加的元素</dd>
</dl>

### 返回值

新长度
